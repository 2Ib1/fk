Wzory Viete'a

Wzorów Viete'a możemy użyć tylko wtedy gdy "Delta" > 0, ponieważ tylko wtedy istnieją pierwiastki funkcji kwadratowej.

Zastosowanie wzorów Viete'a

1. Badanie znaków pierwiastków trójmianu:

a) x1 * x2 < 0 (Obydwa pierwiastki różnych znaków)

b) x1 * x2 > 0 (Obydwa pierwiastki tych samych znaków)

c) x1 * x2 > 0 ∧ x1 + x2 > 0 <=> (Obydwa pierwiastki dodatnie)

d) x1 * x2 > 0 ∧ x1 + x2 < 0 <=> (Obydwa pierwiastki ujemne)

2. Obliczanie wyrażeń zawierających pierwiastki bez obliczania tych pierwiastków:

a) Suma kwadratów pierwiastków:

x12 + x22 = (x1 + x2)2 - 2x1x2

b) Suma odwrotoności pierwiastków:

1/x1 + 1/x2 = (x1 + x2)/(x1 * x2)

c) Suma odwrotoności kwadratów pierwiastków:

1/x12 + 1/x22 = ((x1 + x2)2 - 2x1x2)/(x1 * x2)2

d) Suma sześcianów pierwiastków:

x13 + x23 = (x1 + x2)((x1 + x2)2 - 3x1x2)

e) Suma czwartych potęg pierwiastków:

x14 + x24 = ((x1+x2)2 - 2x1x2)2 - 2(x1x2)2

f) Suma kwadratów pierwiastków pomniejszona o 1:

(x1 - 1)2 + (x2 - 1)2 = (x1 + x2)2 - 2x1x2 + 2
