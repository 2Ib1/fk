Funkcja kwadratowa z parametrem


                1. Badanie liczby rozwiązań w zależności od parametru:
                (m2 - 1)x2 + (m + 1)x + 1 =0
                To równanie jest rónaniem kwadratowym, zatem ilość rozwiązań zależy
                od wyniku "Delty". 
                Pierwszym krokiem w przypadku takiego równania jest rozważenie równania
                liniowego, ponieważ przy "x2" znajduje się parametr.
                I. a = 0 <=> m2 - 1 = 0
                                 m2 = 1
                                |m| = 1
                          m = 1  v  m = -1 
                0x2 + 2x + 1 = 0   0x2 + 0x + 1 = 1
                 x = -1/2                 0 = 1
                Jedno rozwiązaie        Brak rozwiązań
                II. Teraz obliczamy "Deltę" i sprawdzamy jej wynik.
                a ≠ 0 <=> m ∈ R\{-1;1}
                Δ = (m+1)2 - 4 * (m2 - 1) * 1
                Δ = m2 + 2m + 1 - 4m2 + 4
                Δ = -3m2 + 2m + 5
                1) Δ > 0 - dwa różnie rozwiązania 
                   -3m2 +2m + 5 > 0
                   Δm = 4 - 4 * (-3) * 5 = 64, √Δm = 8
                   m1 = 5/3     m2 = -1
                      m ∈ (-1; 5/3) \ {1}
                2) Δ = 0 - jedno rozwiązaie
                   Δ = 0 <=> m ∈ {-1; 5/3} ∧ m ∈ R \ {-1; 1}
                               m ∈ {5/3}
                3) Δ < 0 <=> m ∈ (-∞; -1) v (5/3; + ∞)
                Odp: Dla m ∈ (-1; 1) v (1; 5/3) funkcja ma 2 rozwiązania.
                     Dla m ∈ {5/3; 1} funkja ma jedno rozwiązanie.
                     Dla m ∈ (-∞; -1> v (5/3; + ∞) funkcja nie ma rozwiązań
                
                2. Wyznacznie wartości parametru m, dla których równanie ma dwa
                różne rozwiązanie dodatnie
                (m -2)x2 + (m + 1)x -m = 0
                Pierszym krokiem jaki musimy wykonać jest ustalenie założeń. 
                |-----
                | a ≠	0 = > 1) m ≠ 2 (Rozważamy funkcję liniową) 
                | Δ > 0 (W treści polecenie jest napisane że funjkcja musi mieć
                | 2 rozwiązania, zatem Δ musi być większa od 0)
                | x1 x x2 > 0
                | x1 + x2 > 0 (Obydwa rozwiązania muszą być dodatnie zatem 
                | zarówno suma, jak i iloczyn pierwiastków muszą być dodatnie)
                |-----
                2) Δ > 0
                   Δ = (m + 1)2 - 4 * (m - 2) * (-m) > 0
                       m2 + 2m + 1 + 4m2 - 8m > 0
                       5m2 - 6m + 1 > 0
                   Δm = 36 - 20 = 16, √Δm = 4
                         m1 = 1    m2 = 1/5
                       m ∈ (-∞; 1/5) v (1; +∞)
                3) x1 x x2 > 0
                   c/a > 0
                   -m/m - 2 > 0
                   -m(m - 2) > 0
                   m ∈ (0; 2)
                4) x1 + x2 > 0
                   -b/a > 0
                   -(m + 1)/(m - 2) > 0
                   -(m + 1)(m - 2) > 0
                     m ∈ (-1; 2)
                |-----
                | m ≠ 2
                | m ∈ (-∞; 1/5) v (1; +∞) 
                | m ∈ (0; 2)
                | m ∈ (-1; 2)
                |-----
                W tym momencie należy zsumować wszystkie założenia i "wyciągnąć"
                z nich część wspólną.
                Odp: Zbiór wartości m ∈ (0; 1/5) v (1; 2).
                
                3. Niech x1 i x2 będa pierwiastkami równania. Wyznacz wartości
                parametru "p", dla których zachodzi nierówność x1/x2 + x2/x1 ≥ 6.
                x2 + px + 1 = 0
                Pierszym krokiem jaki musimy wykonać jest ustalenie założeń. 
                |-----
                | a ≠ 0
                | Δ ≥ 0                 =>          1) p2 - 4 * 1 * 1 ≥ 0 
                | x1/x2 + x2/x1 ≥ 6                     p2 ≥ 4 / √
                |-----                                    |p| ≥ 2
                                                       p ≥ 2 v p ≤ -2
                2) x1/x2 + x2/x1 ≥ 6
                   (x1 + x2)/(x2 * x1) ≥ 6
                   ((x1 + x2)2 - 2x1x2)/(x2 * x1) ≥ 6
                   ((-p2) - 2)/1 ≥ 6
                   p2 ≥ 8 / √
                   |p| ≥ 2√2
                   p ≥ 2√2 v p ≤ -2√2
                |-----
                | p ∈ (-∞; -2> v (2; +∞) 
                | p ∈ (-∞; -2√2> v (2√2; +∞)
                |-----
                W tym momencie należy zsumować wszystkie założenia i "wyciągnąć" 
                z nich część wspólną.
                Odp: Zbiór wartości p ∈ (-∞; -2√2> v (2√2; +∞).
                
                4. Wyznacznie parametru m, aby nierówność była prawdziwa dla każdego
                x ∈ R.
                (m + 1)x2 - 3x + 1 > 0
                Parametr stoi przy x2, zatem musimy rozważyć czy nie występuje
                funkcja liniowa.
                I. a = 0
                m + 1 = 0
                m = -1
                0x2 - 3x + 1 > 0
                -3x + 1 > 0
                Sprzeczność
                m = -1 nie spełnia warunków zadania
                W tym momencie musimy ustalić założenia.
                II.
                |-----
                | a > 0 (Nierównośc musi być więskza od 0, zatem a musi być większe 
                | aby ramiona parabeli byli skierowane do góry)
                | Δ < 0 (Nierówność musi być prawdziwa dla każdego x ∈ R, więc "Delta"
                | musi być mniejsza od 0 aby nie było miejsc zerówych a nierówność
                | była zawsze prawdziwa)
                |-----
                1) a > 0
                   m + 1 > 0
                    m > -1
                2) Δ < 0
                   (-3)2 - 4(m + 1) * 1 < 0
                   9 - 4m - 4 < 0
                   -4m < -5
                   m > 5/4
                |-----
                | m > -1
                | m > 5/4
                |-----
                W tym momencie należy zsumować wszystkie założenia i "wyciągnąć" 
                z nich część wspólną.
                Odp: Zbiór wartości m ∈ (5/4; +∞).
                
                5. Wyznacznie wartości parametru m, aby suma kwadratów pierwiastków 
                równania ogiągnęła wartość najmniejszą. 
                x2 + (m + 1)x + 1 = 0
                Parametr nie wsytępuję przy "x2" zatem nie trzeba rozważać funkcji 
                liniowej.
                |-----
                | Δ ≥ 0 ("Delta" większa lub równa 0, aby występowały rozwiązania)
                | x12 + x22 => Minimum
                |-----
                1) Δ ≥ 0
                   (m + 1)2 - 4 * 1 * 1 ≥ 0
                   (m + 1)2 ≥ 4 / √
                   |x + 1| ≥ 2
                   m + 1 ≥ 2   v     m + 1 ≤ -2
                      m ≥ 1             m ≤ -3
                   m ∈ (-∞; -3) v (1; +∞)
                2) x12 + x22
                   (x1 + x2)2 - 2x1x2
                   (-b/a)2 - 2c/a 
                   ( - (m + 1)/1)2 - 2 x 1
                   (m + 1)2 - 2
                   m2 + 2m + 1 - 2
                   m2 + 2m - 1
                   Otrzymaliśmy funkcję kwadratową f(m), której dziedzina 
                   (-∞; -3) v (1; +∞), dzięki czemu jesteśmy w stanie obliczyć jej
                   wartość najmniejszą. 
                   f(m) = m2 + 2m - 1
                   p = -2/2 x 1 = -1 ∉ Df
                   f(-3) = 9 - 6 - 1 = 2
                   f(1) = 2
                Odp: Suma kwadratów będzie najmniejsza gdy m = {-3; 1} i ta wartość 
                wynosi 2.
