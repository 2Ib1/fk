Równania kwadratowe

Kiedy Δ > 0, równanie ma 2 rozwiązania

Kiedy Δ = 0, równanie ma 1 rozwiązanie

Kiedy Δ < 0, równanie nie ma rozwiązań


Etapy rozwiązywania równań kwadratowych:

1.Wypisujemy współczynniki

2.Obliczamy deltę

3.Obliczmy pierwiastki


Przykłady:

a) 2x2 + 3x - 2 = 0

a= 2 b= 3 c= -2

Δ = 32 - 4 * 2 * (-2) = 9 + 16 = 25

√Δ = 5

x1 = (-3 - 5)/4 = -8/4 = -2

x2 = (-3 + 5)/4 = 2/4 = 1/2

Odp.: x ∈ {-2; 1/2}

b) x2 - 4x + 4 = 0

a= 1 b= -4 c= 4

Δ = (-4)2 - 4 * 1 * 4 = 16 - 16 = 0

x0 = 4/2 = 2

Odp.: x = 2

c) x2 - x + 1 = 0

a= 1 b= -1 c= 1

Δ = (-1)2 - 4 * 1 * 1 = -3

Δ < 0
